# Malayalam Automated Essay Scoring & Feedback Generation

An AI-powered system for automatically evaluating Malayalam essays and generating personalized bilingual feedback using NLP, multilingual transformer embeddings, machine learning, and Large Language Models (LLMs).

## Features

- Automated Malayalam essay scoring (0–10 scale)
- Semantic similarity analysis using multilingual embeddings
- XGBoost-based score prediction
- LLM-based personalized feedback generation
- English and Malayalam feedback support
- Reference-answer based evaluation
- Cross-prompt generalization for unseen essay topics

## Tech Stack

- Python
- XGBoost
- Transformers
- Multilingual-E5-Large-Instruct
- Gemma 2B
- Scikit-learn
- PyTorch
- Pandas & NumPy

## Dataset

- 3,000 Malayalam essays
- 150 essay prompts
- Human-annotated scores
- Reference model answers

## Applications

- Automated educational assessment
- Writing skill evaluation
- AI-assisted learning systems
- Teacher support tools
